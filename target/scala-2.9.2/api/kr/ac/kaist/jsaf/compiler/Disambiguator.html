Disambiguator

class Disambiguator extends Walker

Eliminates ambiguities in an AST that can be resolved solely by knowing what kind of entity a name refers to. This class specifically handles the following:

  • Variable/function/label references have unique internal names.
  • All name references that are undefined or used incorrectly are treated as static errors.
